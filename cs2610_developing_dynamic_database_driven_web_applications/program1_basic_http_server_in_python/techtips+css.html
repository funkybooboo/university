Tech Tips with CSS

HTML Elements

<div>

  • Default display: inline
  • Break up content into manageable chunks making it easy to style those chunks.

<pre>

  • Default display: inline
  • Allows you to put text where you would like on the page with no worry of the text moving all to one line, great to use instead of the <p> tag.

<a>

  • Default display: block-level
  • You can make links to other pages, such as another page on your site, or another page out on the internet.

<img>

  • Default display: inline
  • You can put a link to an image file that is saved on your hard drive or out on the internet. If image isn't found that puts a message out on the page.

HTML Character Entities

Copyright Sign: ©

  • Used to show if a name is from a company.

Cent Sign: ¢

  • Used before a number of cents. Such as ¢22.

One Half Fraction: ½

  • Used to show one half of something.

CSS Selectors

type

  • With this selector you can give general attributes to any tag.
  • Such as giving all <p> tags the same background-color.
  • You use them by going to your CSS page and naming the tag you want to use without anything in front of it and then brackets.
  • Any CSS properties in the brackets will be applied to all of the tags of that type.
                    tag {

                    }

.class

  • With this selector you can give multiple tags a class name identifier.
  • Give the tags you want to have the same CSS attributes the same class name and they all will be effected.
  • You use them by going to your CSS page and naming the class name that you gave to your tags and then brackets.
  • Any CSS properties in the brackets will be applied to all of the tags of that class.
                    .name {

                    }

#id

  • With this selector you can give a tag an id with a name identifier.
  • Give the tag you want to have a set of particular CSS attributes.
  • You use them by going to your CSS page and naming the id name that you gave to your tag and then brackets.
  • Any CSS properties in the brackets will be applied to all of the tags of that id.
                    #name {

                    }

:nth-child

  • With this selector you can specify how many children down from another selector you want to gave attributes.
  • Use them by naming the type, id, or class you are talking about than a colon then nth-child() with a An+B going into the parentheses
  • A is an integer size
  • B is an integer offset
  • n is all non-negative integers, starting from 0.
Got my info from MDN 
                    attribute:nth-child(An+B) {

                    }

CSS Properties

padding

  • With this property you can select how much spacing you would like the tag to have from other tags or the edge of the screen.
  • Some possible values include: 15px, inherit, length, ect...

background-color

  • With this property you can color the background in the selected attribute.
  • Some possible values include: yellow, black, white, ect...

color

  • With this property you can color the text in the selected attribute
  • Some possible values include: red, blue, green, ect...
